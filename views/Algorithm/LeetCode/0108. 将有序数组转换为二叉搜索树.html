# 108. 将有序数组转换为二叉搜索树

钟 波

# 题目描述

题目传送门：108. 将有序数组转换为二叉搜索树 (opens new window)

给你一个整数数组 nums ，其中元素已经按 升序 排列，请你将其转换为一棵 高度平衡 二叉搜索树。

高度平衡 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1 」的二叉树。

示例 1：

输入：nums = [-10,-3,0,5,9]
输出：[0,-3,9,-10,null,5]
解释：[0,-10,5,null,-3,null,9] 也将被视为正确答案：

示例 2：

输入：nums = [1,3]
输出：[3,1]
解释：[1,3] 和 [3,1] 都是高度平衡二叉搜索树。

提示：

  • 1 <= nums.length <= 10^4

  • -10^4 <= nums[i] <= 10^4

  • nums 按 严格递增 顺序排列

# 题解

注意题目要求我们要构造的二叉树是 高度平衡 的，因此如何选取根节点就比较重要了，对于一个具有奇数个数的数组 nums，中间位置是固定的只有一个，但对于一个具有偶数个数的数组 nums，中间位置有两个元素，不论选取哪个元素作为根节点都是可以的，主要取决于在nums数组中通过两个边界来确定中间位置的写法，下面选取中间位置左边的数字作为根节点构造二叉搜索树。

class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return buildTree(nums, 0, nums.length - 1);
    }
    
    public TreeNode buildTree(int[] nums, int left, int right) {
        // 当 left>right 时，平衡二叉搜索树为空
        if (left > right) return null;
        // 这种写法防止整型溢出
        int mid = left + (right - left) / 2;
        TreeNode node = new TreeNode(nums[mid]);
        node.left = buildTree(nums, left, mid - 1);
        node.right = buildTree(nums, mid + 1, right);
        return node;
    }
}
